# 代码规范

#### html

1. 拒绝将大量样式写入某一个标签中

```html
<div style="font-size: 18px; color: #00bc8c; text-align: center"></div>
```

应该修改为：

```css
.demo{
	font-size: 18px; 
    color: #00bc8c; 
    text-align: center
}
```

```html
<div class="demo"></div>
```

##### 拒绝代码和采取方案：

```js
1. 多if
if(a==0){
    if(b==0){
        if(c==0){
            if(d==0){
               
            }
        }
    }
}
`解决方案：`
当使用多个if嵌套的时候会使代码的可读性变差，可以采用 `!==` + `return`

if(a!==0 && b!==0 && c!==0 && d!==0){
    return
}

console.log(xx)

2.定义元素，但不使用
var element;
var tempData;
var data;
var a;
var b;

a=1;
data=0;

`解决方案：1.让初始值为努力了;当元素是null值的时候，未使用的值会被释放。2.将不需要使用的元素删掉`
var element=null;
var tempData=null;
var data=null;
var a=null;
var b=null;

a=1;
data=0;
```
